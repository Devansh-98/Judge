In this problem, you will be given an array A of size N. The array will be 0-indexed and each element of the array will be a positive integer not greater than 9. You will be asked Q queries on this array. In each query you will be given 3 integers l,r,x. You have to determine if the element x is present in the range A[l],A[l+1]....A[r]

Input:

First line of each test case contain the integer N, the size of the array. Second line contain N space separated integers denoting the elements of the array Next line contains the integer Q Q lines follow where each of them contain 3 space seperated integers l,r,x.

Output:

For each testcase, output Q lines corresponding to the Q queries. Print "yes" if x is present in the given range, otherwise print "no".

1≤Q≤10^5
